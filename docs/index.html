Features

Genome Upload

Import pet genome .txt files directly from Project Gorgon. Drag-and-drop or use native file dialogs.

Gene Visualization

Interactive color-coded grid showing every chromosome and gene effect. Switch between attribute and appearance views.

Gene Editor

Edit dominant and recessive gene effects per chromosome. Define custom gene templates for any species.

Attribute Stats

See summarized attribute scores (Toughness, Ferocity, Intelligence, etc.) with counts and breakdowns.

Multi-Species

Supports BeeWasp (10 chromosomes) and Horse (48 chromosomes) with species-specific attribute configs.

Local & Private

All data stored locally in SQLite. No accounts, no cloud, no tracking. Your pet data stays on your machine.

Installation Guide

macOS

  1. Download the .dmg file above
  2. Open the .dmg and drag Gorgonetics into your Applications folder
  3. On first launch, macOS will block the app because it's unsigned. Right-click the app → Open, then click Open in the dialog
  4. Alternatively, go to System Settings → Privacy & Security and click "Open Anyway"

Windows

  1. Download the .exe installer above
  2. Run the installer — it will guide you through the setup
  3. If Windows SmartScreen shows a warning, click "More info" then "Run anyway"
  4. Launch from the Start Menu or desktop shortcut
Note: Gorgonetics stores all data locally in a SQLite database. No account or internet connection is required. On first launch, the app loads demo gene templates and sample pets so you can explore immediately.
